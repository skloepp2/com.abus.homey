# ABUS Z-Wave

Adds support for ABUS Z-Wave devices
- CFA3010
- SHBW10000
- SHHA10000
- SHMK10000
- SHRM10000
- SHSG10000
- SHWM10000

This is a port of https://homey.app/de-de/app/com.abus/ABUS/ to the new SDK 3 to add support for Homey Pro (2023).
Special thanks to Martin P for the initial version! -> https://homey.app/de-de/apps/author/5dcc1c8d63ab870c55c7f8d9/
